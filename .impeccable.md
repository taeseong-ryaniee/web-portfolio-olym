# Design System — web-portfolio-olym

## Design Context

### Users
- **Primary audience**: 클라이언트/의뢰인 — 프리랜서 또는 에이전시에 프로젝트를 의뢰하려는 사람들
- **Context**: 여러 프리랜서/에이전시를 비교 평가하는 상황에서 방문. 빠르게 역량을 판단하고 신뢰를 형성해야 함
- **Job to be done**: 이 프리랜서/에이전시가 자신의 프로젝트를 맡길 수 있는 실력과 신뢰성을 갖추었는지 확인

### Brand Personality
- **3 Keywords**: 전문적 (Professional) · 신뢰 (Trustworthy) · 깔끔 (Clean)
- **Voice**: 과시하지 않되, 작업물이 스스로 말하게 하는 자신감 있는 톤
- **Emotional goal**: 방문자가 "이 사람/팀에게 맡기면 안심이 되겠다"는 확신을 느끼게 하기

### Aesthetic Direction
- **Visual tone**: 에디토리얼/매거진 — 타이포그래피 중심, 여백 활용, 잡지 같은 레이아웃
- **Theme**: 라이트 모드 전용
- **Color palette**: 뉴트럴 기반 (웜 그레이, 오프화이트)에 하나의 정제된 액센트 컬러. 에디토리얼 느낌에 맞는 절제된 팔레트 — 과한 색상 사용 금지
- **Typography**: 디스플레이용 세리프 + 본문용 산세리프 조합. 잡지 에디토리얼 분위기
- **Content focus**: 프로젝트 작업물(케이스 스터디) 중심
- **Anti-references**: AI 슬롭 (사이안+다크, 글라스모피즘, 그라디언트 텍스트), 지나치게 장식적인 디자인, 카드 그리드 반복

### Design Principles
1. **작업물이 주인공이다** — 인터페이스는 작업물을 돋보이게 하는 프레임이지, 그 자체가 주인공이 아니다
2. **절제된 우아함** — 모든 요소는 존재 이유가 있어야 한다. 장식을 위한 장식은 없다
3. **타이포그래피가 구조를 만든다** — 크기, 굵기, 간격으로 명확한 시각적 위계를 형성
4. **여백은 디자인이다** — 빈 공간은 비어 있는 것이 아니라 의도적으로 설계된 것
5. **신뢰는 디테일에서 온다** — 정렬, 일관성, 세심한 타이포그래피가 전문성을 증명

### Accessibility
- **WCAG AAA 준수** — 최고 수준의 접근성
- 7:1 이상의 텍스트 대비율
- reduced-motion 미디어 쿼리 지원
- 시맨틱 HTML, 키보드 내비게이션, 스크린 리더 완전 지원
- 포커스 인디케이터 명확하게 표시

### Technical Notes
- 프리랜서/에이전시 포트폴리오 — 프로젝트 케이스 스터디 중심
- 라이트 모드 전용 (다크 모드 불필요)
- 에디토리얼 레이아웃: 비대칭, 다양한 그리드 변형, 의도적 여백
